# Ember-typewriter

<a href="https://shipshape.io/"><img src="http://i.imgur.com/bU4ABmk.png" width="100" height="100"/></a>

Ember-Typewriter is an Ember addon that wraps Typed.js.

## Installation

`ember install ember-typewriter`

## Usage

```hbs
{{ember-typewriter 
class="element" 
contentType="html" 
strings=model.strings 
tagName="span"}}
```
